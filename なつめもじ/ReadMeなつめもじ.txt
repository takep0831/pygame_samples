　　　　　　　　（2025年/1月）

フォントを気に入って下さってありがとうございます。
必ず下記をお読みになってからお使い下さい。
使い方が分からない場合は、ご使用を控えて下さい。


◆「なつめもじ」「なつめもじ抑」はWindows用のTrueTypeフォントです。
　　武蔵システム（https://opentype.jp/）のソフト、TTEditにて作成しています。
　　iPadなどApple製品でもお使いいただけます。


◆フリーフォントですが、著作権は放棄していません。
　再配布・販売・改造は絶対にしないでください。


◆シフトJIS漢字コード表の全ての漢字を書くことができます。
　一部の記号（┨など）は絵文字になっています。
　一部の記号と一部のキリル文字は未作成のため空欄になります。


◆文字デザインとして、ハネやハライを省略・簡素化したり、
　バランスを変えている文字があります。


◆全角数字は、日本語と大きさが揃うように作りました。
　日本語の文章内で数字を使う場合は、全角数字にすると自然な感じになります。


◆Webフォント用にファイル変換してお使いになることは可能です。
　変換後のフォントデータを配布・他人に譲渡することはしないでください。
　禁止事項の再配布に当たります。


◆印刷物（オフラインでの利用全般）でも、
　商用（チラシ・ポスター等）・非商用（個人のハガキ等）を問わずお使いいただけます。
　事前または事後のご連絡は不要ですが、使用報告はとても嬉しいです。
　書籍・商品でお使いの場合、見本をいただけるとすごく嬉しいです。
　もちろんお知らせだけでも嬉しいです！


◆スマートフォンのアプリに組み込んだり、
　YouTube・ニコニコ動画等の作品にもご自由にお使いいただけます。
　使用の際、「なつめもじ」の表記や当サイトへのリンクはとても嬉しいですが、
　リンクなしでも問題ありません。ぜひフォントをご活用ください。


◆webサイト・SNS・書籍等で当サイトのフォントをご紹介下さる時は、
　必ずトップページ（https://apricot.ciao.jp/）にリンクを貼ってください。
　トップページ以外はURLが変わることがあります。
